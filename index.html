クラウド自動同期の設定

GitHub Personal Access Token（PAT）を設定すると、Mac・iPhoneで自動的にデータが同期されます。

PATの作成方法：
1. こちらをタップ（GitHub → Settings → Tokens）
2. Expiration を「No expiration」に設定
3. 「Generate token」をクリックしてトークンをコピー
